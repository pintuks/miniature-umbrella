A premium, mobile-first Valentine Week micro-site for Sweety.

## Run locally

Use any static server. Example:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000` in your browser.

## Customization

- Replace gallery image placeholders by editing `galleryItems` in `script.js`.
- Update day copy, dates, and accents in the `valentineWeekDays` array.
- Change countdown target settings in the `settings` object (timezone locked to Asia/Kolkata).
- Replace `assets/og-placeholder.svg` with a custom Open Graph image.
